# GenAI Academy

Interactive course on Generative AI, Prompt Engineering & Workflows

## 🎓 Lessons

- [Main Course (Lessons 1-4)](genai-course-final.html) - Glossary, AI Fundamentals, Prompt Engineering, Activities
- [Choosing the Right AI Tool](lesson-5-choosing-ai-tool.html) - Compare Claude, ChatGPT, Gemini
- [Common Mistakes](lesson-6-common-mistakes.html) - Avoid hallucinations, data privacy, poor prompts
- [AI Ethics](lesson-7-ai-ethics.html) - Bias, fairness, responsibility
- [AI in Your Workflow](lesson-8-ai-workflow.html) - Real-world applications by role
- [Building Workflows](lesson-9-building-workflows.html) - Automation & scaling

## ✨ Features

✅ 10 Interactive 3D Flashcards
✅ Side-by-side Comparison Tables
✅ Click-to-Reveal Mistake Cards
✅ Persona-Based Learning (4 roles)
✅ Real-time Quizzes with Feedback
✅ Mobile Responsive Design
✅ Zero Dependencies (Vanilla JS)

## 🛠 Tech Stack

HTML5 | CSS3 | Vanilla JavaScript | Responsive Design
